# 🏏 Abhishek Sharma Performance Analysis (IPL) – Power BI Dashboard

## 📌 Project Overview

This project presents a comprehensive **data analytics dashboard** analyzing the performance of **Abhishek Sharma** in the IPL using **Power BI**.

The dashboard provides insights into batting performance, consistency, match impact, and strengths against different bowling types.

---

## 🎯 Objectives

* Analyze batting performance across seasons
* Evaluate consistency (50s, 100s, average)
* Measure scoring ability (Strike Rate, Boundary %)
* Compare performance against **Spin vs Pace**
* Identify match-winning contributions

---

## 📊 Key Metrics

* Total Runs
* Balls Faced
* Strike Rate
* Batting Average
* Total 4s & 6s
* Boundary Percentage
* Highest Score
* 50+ Scores
* 100+ Scores
* Match Winning Contribution

---

## 📈 Dashboard Visuals

* 📊 **Season-wise Runs**
* 📊 **Strike Rate across Match Phases**
* 📊 **Performance vs Spin & Pace**
* 🍩 **Boundary Distribution (4s vs 6s)**
* 🥧 **Modes of Dismissal**
* 📊 **Win vs Loss Contribution**

---

## 🧠 Key Insights

* Strong performance in **Powerplay overs**
* Higher strike rate against **pace bowlers** compared to spin
* Majority of runs scored through **boundaries (4s & 6s)**
* Consistent contributions in match-winning scenarios
* Dismissals mainly through **caught** indicating aggressive gameplay
